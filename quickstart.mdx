There is no single quickstart for BasedMining — different audiences plug in differently. Pick the path that matches you.

## I want to mine with my own hardware

You will need:

- An ASIC or compatible mining device (Apollo, Bitaxe, S19, etc.)
- A Bitcoin wallet address you control
- A worker name (anything you want — `home1`, `garage`, `loud-rig`)

Go to **[Connect Your Rig](/miners/connect-your-rig)** for the full walkthrough.

## I want to mine without hardware

You will need:

- A wallet that supports Base (Rainbow, Coinbase Wallet, MetaMask, etc.)
- A small amount of ETH on **Base Sepolia** for the testnet phase
- About two minutes

Go to **[Mint a BasedPool NFT](/onchain/mint)** for the walkthrough.

## I'm an agent or building one

The agent surface is **not yet live**. We're publishing the design now so agent developers can start integrating against the spec.

Go to **[For Agents](/agents/overview)** to see the planned interface.
