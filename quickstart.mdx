There is no single quickstart — different audiences plug in differently. Pick the path that matches you.

## I want to mine with my own hardware

You'll need:

- An ASIC or compatible mining device
- A Bitcoin wallet address you control
- A worker name (any label — `home1`, `garage`, `loud-rig`)

→ [**Connect Your Rig**](/miners/connect-your-rig) for the full walkthrough.

## I want hashpower without hardware

You'll need:

- A wallet that supports Base (Rainbow, Coinbase Wallet, MetaMask, Frame, etc.)
- ETH, USDC, or cbBTC on **Base mainnet** to pay for the mint
- About two minutes

→ [**Mint Hash**](/onchain/mint-hash) for the walkthrough.

## I'm an agent or building one

The agent surface is **in development**. The spec is published so agent builders can integrate against it.

→ [**For Agents**](/agents/overview) for what's planned.
