## What we're building

A pay-per-call mining-optimization layer for AI agents. The idea:

- An agent has hashrate (its own, or rented)
- BasedMining offers programmable services that improve what that hashrate earns
- Agents pay per service call using **x402** — the HTTP-native stablecoin payment standard

No accounts, no API keys, no negotiation. The agent makes a request, the response is gated on a 402 payment, the agent pays, and the call completes. Same pattern an LLM uses to call any other tool.

## Why agents?

Bitcoin mining is a 24/7 optimization problem: temperature, electricity rates, pool selection, firmware tuning, hashrate routing. Most home miners don't optimize most of those most of the time, because nobody has the bandwidth.

Agents do. An always-on agent can:

- Monitor pool conditions
- Reroute hashrate when economics shift
- Tune firmware to match power costs in real time
- Make per-block decisions a human can't

We are building the **service surface** that agents will call to do this — and a payment layer that lets them pay only for what they use.

## How payment works

We're using **x402** — an open standard for HTTP-native stablecoin payments.

The flow:

1. Agent makes a request to a BasedMining service endpoint
2. Server responds with `402 Payment Required` and the price
3. Agent submits a payment payload (USDC on Base)
4. Server verifies, executes the service, and returns the result

Agents pay per call. No subscriptions, no API keys to rotate, no minimum spend. If you're building an agent on standard MCP/HTTP rails, you already have the integration.
